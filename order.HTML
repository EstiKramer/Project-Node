API Order Operations

שיטה כתובת פרמטרים ב-body מה עושה סוגי שגיאות אפשריות
GET /api/order/ אין מחזיר את כל ההזמנות
  • 500 - שגיאה בשרת
  • 400 - בעיה בביצוע הבקשה
GET /api/order/user/:userId אין מחפש הזמנות לפי ID של משתמש
  • 404 - לא נמצאו הזמנות למשתמש הזה
  • 400 - ID לא תקני
  • 500 - שגיאה בשרת
POST /api/order/ 
{
    "userId": "ID של המשתמש",
    "products": [
        {
            "name": "שם המוצר",
            "price": 100,
            "sizes": 42,
            "imagePuth": "נתיב התמונה"
        }
    ],
    "address": "כתובת למשלוח",
    "pricesend": 10,
    "finallprice": 110
}
יוצר הזמנה חדשה
  • 404 - חסר פרמטרים (userId, products או address)
  • 400 - כתובת לא תקינה
  • 400 - מחיר לא תקין (שליחה או סה"כ)
  • 404 - לא נמצא משתמש עם ID זה
  • 500 - שגיאה בשרת
PUT /api/order/:id אין מעביר הזמנה לסטטוס "נשלחה"
  • 404 - לא נמצאה הזמנה עם ה-ID הזה
  • 500 - שגיאה בשרת
DELETE /api/order/:id אין מוחק הזמנה לפי ID
  • 404 - לא נמצאה הזמנה עם ה-ID הזה
  • 500 - שגיאה בשרת
